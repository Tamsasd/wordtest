auto-submit:
automatically evaluates answer when the charactercount reaches the correct answer's length.
-------
case sensitivity:
if set off, ignores if a character is lower or upper case.

for example, if the guess is "abc" and the correct answer is "Abc", the guess will be correct too.
-------
skip evaluation:
automatically moves forward to the next word without showing the wrong answer's evaluation panel.
-------
ignore accents:
if set on, ignores if a character is accented or not.

for example, if the guess is "abc" and the correct answer is "ábc", the guess will be correct too.
-------
mode:
practice mode: goes on infinitely, or until a setting or the user stops it.
can ask words multiple times

exam mode: goes on until there's no more words left, or until a setting or the user stops it.
asks each word exactly once.
-------
order:
controls the order in which the words will be asked.
-------
countdown:
triggers the summary page after the set amount of seconds elapsed.
-------
stop-percentage:
triggers the summary page after the set amount of percentage is reached.
can't be used in exam mode.
-------
colors:
modifies the appearance of the page.
